# GoIT Advanced JS - Homework 06: Typing & Generics

This project contains solutions for Topic 10, focusing on basic types and the advanced implementation of Generics and Utility Types in TypeScript.

##  Project Configuration
- **TypeScript**: Version 5.1.5.
- **Compiler**: Configured with strict null checks and source maps for debugging.
- **Build System**: Automated via GitHub Actions to compile and deploy to GitHub Pages.

##  Tasks
- **Basic Types**: Proper typing for primitives, tuples, and enums.
- **Generics**: Reusable functions and interfaces using generic constraints.
- **Utility Types**: Applied `Partial`, `Pick`, `Omit`, and `Record` for efficient data transformation.
